this is filtr

filtr is a command-line tool for applying filters to pictures.

filtr is a wrapper around a lot of smaller pieces most notably ImageMagick or GraphicksMagick, neither of which are very small. Individual filters are defined as "recipes" (which can be invoked on their own) and all the filtr script does is some basic housekeeping around calling them, including things like transferring EXIF data from a source image to the newly filtered version.
